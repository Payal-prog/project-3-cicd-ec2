Built an automated CI/CD pipeline using GitHub Actions to deploy a web application to an AWS EC2 instance. The pipeline enables seamless code integration and automated deployment via SSH, eliminating manual deployment steps and improving release efficiency.

This project demonstrates hands-on experience with CI/CD workflows, cloud infrastructure, Linux servers, and real-world deployment practices.

🚀 Automated CI/CD & Cloud Orchestration

- CI/CD: Engineered a multi-stage GitHub Actions pipeline to automate linting, testing, and deployment, reducing manual intervention to zero.

- Cloud Infrastructure: Provisioned and configured AWS EC2 environments, ensuring high availability and optimized resource allocation.

- Security & Secrets: Implemented secure SSH/Rsync deployment patterns utilizing GitHub Secrets and IAM best practices to eliminate credential exposure.

- Reliability: Reduced Deployment Lead Time (DLT) by automating the build-to-production lifecycle, facilitating a fail-fast feedback loop for developers.
